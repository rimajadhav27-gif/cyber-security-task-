TASK 1: FILE INTEGRITY CHECKER

Objective:
To monitor file changes by calculating and comparing hash values to ensure file integrity.

Technologies Used:
Python
hashlib, os, json, sys

Features:
- Creates baseline hash values
- Detects modified, added, or deleted files
- Helps prevent unauthorized file changes


TASK 2: WEB APPLICATION VULNERABILITY SCANNER

Objective:
To detect common web vulnerabilities such as SQL Injection and Cross-Site Scripting (XSS).

Technologies Used:
Python
requests, BeautifulSoup, re

Features:
- Scans web pages
- Tests input fields with payloads
- Identifies potential vulnerabilities


TASK 3: PENETRATION TESTING TOOLKIT

Objective:
To build a modular toolkit for penetration testing for educational purposes.

Technologies Used:
Python
socket, threading, itertools

Modules:
- Port Scanner
- Brute Force Module

Note:
This tool is strictly for ethical and educational use only.


TASK 4: ADVANCED ENCRYPTION TOOL (AES-256)

Objective:
To encrypt and decrypt files securely using AES-256 encryption.

Technologies Used:
Python
cryptography, os

Features:
- Secure key generation
- File encryption
- File decryption
- Protects sensitive data
